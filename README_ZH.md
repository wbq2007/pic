# 抖音内容审核自动化工具

**重要提示：本工具仅用于个人研究目的，请严格遵守抖音平台的服务条款和法律法规。**

## 快速开始

### 第一步：安装环境
1. **双击 `install.bat`** 自动安装所有依赖
2. 或手动安装：
   ```bash
   pip install -r requirements.txt
   playwright install chromium
   ```

### 第二步：测试环境
**双击 `test.bat`** 验证环境是否配置正确

### 第三步：运行自动化
**双击 `run_playwright.bat`** 启动自动化脚本（推荐使用Playwright版本）

## 功能特点

1. **智能评论搜索**：自动查找包含指定关键词的评论
2. **一键举报**：对违规评论进行批量举报
3. **内容过滤**：将不感兴趣的视频标记为"不感兴趣"
4. **灵活配置**：通过配置文件自定义关键词和设置

## 配置文件说明

编辑 `config.yaml` 文件进行配置：

```yaml
# 关键词列表（用于匹配评论）
keywords:
  - "女权主义"
  - "女权"
  - "表情包里带有女权主义"

# 自动化设置
settings:
  headless: false  # 是否显示浏览器界面
  max_videos: 10   # 最大处理视频数
  scroll_count: 5  # 滚动加载次数
```

## 使用流程

### 首次运行
1. 运行 `run_playwright.bat`
2. 浏览器自动打开抖音网站
3. **手动登录**你的抖音账号
4. 按提示继续自动化流程

### 日常使用
1. 首次登录后会自动保存登录状态
2. 后续运行无需重新登录
3. 根据需求调整 `config.yaml` 中的关键词

## 技术架构

### 推荐方案：Playwright版本
- ✅ 执行速度快，稳定性好
- ✅ 内置移动端设备模拟
- ✅ 现代化API设计

### 备选方案：Selenium版本
- ✅ 兼容性好，资源丰富
- ❌ 执行速度较慢
- ❌ 需要额外配置

## 故障排除

### 常见问题

#### 1. 无法登录抖音
- 检查网络连接
- 确认账号密码正确
- 可能需要处理短信验证码

#### 2. 找不到页面元素
- 抖音页面结构可能已更新
- 需要调整代码中的元素选择器
- 增加页面等待时间

#### 3. 运行速度太慢
- 适当增加操作间隔时间
- 减少同时处理的视频数量
- 使用性能更好的计算机

#### 4. 账号被封禁
- 立即停止自动化操作
- 联系抖音客服申诉
- 降低后续操作的频率

### 调试建议
1. 先在小范围内测试
2. 使用非无头模式观察流程
3. 逐步增加自动化程度
